BlogBundle
==========

# Description
---------------------------------------

Tabernicola's Blog bundle for symfony2

# Installation via composer
---------------------------------------

Add tabernicola's blog to your `composer.json`:

#### 1.1   Symfony 2.3.X
```json
"require": {
    "tabernicola/blog": "dev-master"
},
```

Then run `php composer.phar update` command.
